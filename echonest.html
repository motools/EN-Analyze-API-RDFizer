Echonest Analyze API to Music Ontology RDF

A really simple Echonest Analyze XML to Music Ontology RDF XSLT.

Original file: analyze-example.xml Transform: echonest.xsl Results: analyze-example.rdf



The outputted RDF is designed using the Music Ontology, the Audio Features ontology and a small Echonest ontology extending the latter.
